# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-08

### Added

* Initial project structure
* FastAPI backend
* Streamlit frontend
* XGBoost regression model
* Label encoding pipeline
* Model training workflow
* Prediction API endpoint
* Health check endpoint
* Model information endpoint
* Structured logging
* Documentation

### Changed

* Improved API response formatting
* Added prediction response timing
* Enhanced frontend validation

### Fixed

* Encoder mismatch issues
* Feature name mismatch errors
* Model serialization issues
* Neighborhood encoding errors
* Logging directory configuration

### Security

* Added input validation
* Added exception handling
* Protected API inference pipeline
